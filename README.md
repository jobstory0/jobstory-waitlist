# crediblitz — AI Resume & Portfolio Generator

crediblitz is an AI-powered platform that helps job seekers create **ATS-friendly resumes** and **professional portfolios** with minimal effort.

## Product Vision

crediblitz aims to become a **complete career profile platform** where users can:

- Build professional resumes
- Generate personal portfolios
- Improve career profiles using AI
- Share professional identity easily

## Key Features

- **Resume Builder**: Create ATS-friendly resumes in a single click.
- **Portfolio Generator**: Automatically deploy personal websites using GitHub Pages.
- **AI Career Tools**: AI-powered resume and portfolio creation for professionals.

## Tech Stack

- **Frontend**: Next.js, React, TypeScript
- **Styling**: Tailwind CSS
- **Authentication**: BetterAuth Integration

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
